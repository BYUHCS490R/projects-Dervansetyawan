My Answers

What is normal flow in HTML?

Normal flow is the default way the browser arranges elements on a page. Block elements stack vertically, one after another, while inline elements line up horizontally within text. It’s how the layout appears before any special CSS positioning is applied.

What does float do? Why do you need to clear the float?

The float property moves an element to the left or right, allowing text or inline elements to wrap around it, like wrapping text around an image. You need to “clear” floats because floated elements are removed from the normal flow, which can cause layout issues like parent containers collapsing. Clearing makes elements appear properly below floated ones.

What do absolute, fixed, and relative do?

Relative moves an element slightly from its normal spot while keeping its space; absolute used to place an element exactly at a specific position, ignoring the normal flow of the page; and fixed positions it relative to the browser window, staying in place even when the page scrolls.

How can you make nice navigation buttons for mobile devices?

To make mobile-friendly navigation, use responsive design with flexbox or grid, large touchable buttons, and media queries to adjust layout on smaller screens. You can also use a hamburger menu to save space and make navigation easier on mobile devices.
