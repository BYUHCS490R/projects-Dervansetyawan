I learned that Flex in CSS helps to arrange items easily in a row or column and makes the layout flexible. By using properties like display: flex, justify-content, and flex-wrap, I can control how elements are spaced and aligned on the page. I also learned that I can change the order of items, like moving one box before others using order. Flex makes it easier to create clean, responsive designs without using complicated positioning.
