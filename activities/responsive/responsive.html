My Answers

The viewport meta tag is important because it helps web pages look good on all types of devices, like phones, tablets, and computers. It tells the browser how to control the page’s size and zoom level. Without it, a website might look too small or too big on mobile screens. Using the viewport tag makes the page fit nicely and easier to read on any screen size.

We can adjust the CSS for printing by using a special CSS rule called media print. We can set different styles that only apply when someone prints the page.

We would want to adjust the CSS for printing when the webpage has things that don’t need to show up on paper, like buttons, ads, or navigation bars. Changing the print CSS makes printed pages look more professional and saves ink and paper.
