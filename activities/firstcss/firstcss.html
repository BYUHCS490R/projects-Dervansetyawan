Questions :
  1. Explain the difference between inline, embedded, and external CSS.
    2. Inline CSS : CSS written directly inside the HTML tag using the style attribute, only affects one element. Embedded CSS : CSS written inside a style tag within the HTML files head section., affects the whole page but only works for that file. External CSS : CSS stored in a separate .css file, then linked to the HTML file using link.
  2. Explain why you should only use external CSS.
    3. External CSS is the best way to style a website because it keeps the design rules in a separate file, making the HTML clean and easy to read. With one CSS file, you can control the look of many webpages at once, which saves time and ensures a consistent design across the whole site. If you ever want to make changes, like updating a color or font, you only need to edit the CSS file once, and the change will appear everywhere.
  3. What is a selector used for?
    4. Selector is used to target HTML elements so you can apply styles to them.
