Internet is a big network that connects computers all around the world. It lets people share information, send messages, watch videos, play games, and more. Web (World Wide Web) is a part of the Internet. It is made of websites and web pages we can visit using a browser (like Chrome or Firefox).

Git is a tool that helps people keep track of changes in their code. It saves different versions of files so you can go back if something breaks. GitHub is a website where people can store and share their Git projects online. It makes it easy for teams to work together on code. We using GitHub to save our code safely online, work together with teammates, share our projects with others, keep track of changes and updates.
