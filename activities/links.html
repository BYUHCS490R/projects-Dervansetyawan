Links

Root is the top most folder in a website file structure. All other folders and files are stored inside this root folder.

The relationship between folder and file names and the path : A path shows the exact location of a file within folders. The path must match the folder and file names exactly.

A default document is the webpage that loads automatically when a user visits a website or folder without specifying a file name. Example : index.html.

List the 3 rules with relative links :

  1. Same folder : just the file name.
  2. Inside a subfolder : folder name + / + file name.
  3. Go up one level : .. / moves up one folder.

Links can appear anywhere in a page (paragraphs, footers, sidebars, etc.) not only inside nav tag.

Absolute

Absolute Link : Includes the full URL with domain name.

Relative

Relative Link : Points to a file relative to the current file location in the folder structure.

Benefit of use relative link :

  • Easier to move the whole website to another server/domain.
  • Keeps links shorter and easier to manage.
