AJAX is a way for a website to send or get data from a server without reloading the whole page. It makes the page feel faster and smoother, like when we send a message or click “like” and the page updates instantly.

We put JavaScript in a separate file to keep the code clean and organized. It makes the website easier to read, easier to fix, and it loads faster because the browser can save (cache) the file.
