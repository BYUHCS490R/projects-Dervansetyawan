Internet

Internet is a global system network that connect device all people around the world so they can easy to transfer information, and communicate.

Web

Web (World Wide Web) is the system of connected documents as a part of internet that include picture, video, and sound, that can be searched uses web browsers. For example, when you open youtube in crome.

How are the Internet and Web different?

The Internet is a global network (infrastructure, like roads) while the Web is a collection of information that is accessed via the Internet (cars that traveling on those road).

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) used for the representation of text such as symbols, letters, and digits in computers. This is old character encoding (7 bit, later 8 bit) and only support 128 or 256 characters use English language while Unicode is the universal character encoding used to process, store and facilitate the interchange of text data in any language.

What is UTF-8

UTF-8 is an encoding system for Unicode or a way to store Unicode characters and most common on the web today.

URL

URL is Uniform Resource Locator, the full address of website that we want opened.

Domain Name

Domain Name is a memorable, easy to use we address that points to a website's IP address on the internet.

HTTP

HTTP(Hypertext Transfer Protocol) is the protocol for transferring web pages but insecure and vulnerable to interception.

HTTPS

HTTPS (HTTP Secure) is the protocol for transferring web pages uses SSL/TLS encryption to secure that data.

IP Address

IP Address is unique number in every device on internet, similar to a home address for mail.

DNS

DNS (Domain Name System) is the phonebook of the internet that convert domain name into IP address.

W3C Recommendations

W3C Recomendations (World Wide Web Consortium) is sets official web standards, like official rulebooks for how the web should work that decided by a group of experts, companies, and organizations and they make sure websites look and behave the same across browsers and the web is accessible to everyone.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) is data format, self-descriptive, used for configuration & data exchange; XHTML (Extensible HTML) is a stricter version of HTML written with XML rules; HTML (HyperText Markup Language) is standard language for web pages; HTML5 (Latest version) is HTML that supports video, audio, canvas, offline apps, semantic tags.
